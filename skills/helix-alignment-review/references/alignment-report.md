# Alignment Report Structure

Use this section order:

1. Review Metadata
2. Scope and Governing Artifacts
3. Intent Summary
4. Planning Stack Findings
5. Implementation Map
6. Gap Register
7. Traceability Matrix
8. Review Bead Summary
9. Execution Beads Generated
10. Execution Order
11. Open Decisions
